## Overview

### The challenge

Users should be able to:

- Navigate through a modern, responsive portfolio website with smooth animations
- Read technical blog posts with syntax highlighting and responsive layouts
- View detailed project case studies with live demos and source code
- Access a curated list of development resources and tools
- Contact me through a secure, validated contact form
- Switch between light and dark themes for comfortable viewing

## My process

### Built with

- Modern tech stack and best practices:

  - [Next.js 15](https://nextjs.org/) - React framework with App Router
  - [TypeScript](https://www.typescriptlang.org/) - Type-safe development
  - [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS
  - [shadcn/ui](https://ui.shadcn.com/) - Accessible UI components
  - [Motion](https://motion.dev) - Smooth animations
  - [MDX](https://mdxjs.com/) - Enhanced Markdown for blog posts
  - [EmailJS](https://www.emailjs.com/) - Serverless contact form
  - [pnpm](https://pnpm.io/) - Fast package manager

- Development practices:
  - Mobile-first responsive design
  - Semantic HTML structure
  - Component-driven architecture
  - Progressive enhancement
  - Accessibility (a11y) best practices
  - Performance optimization
  - View transitions between pages

### Key Features

#### Blog Platform

- MDX-powered blog posts.
- Rich content support including code blocks and images
- Responsive layout for optimal reading experience

#### Project Showcase

- Detailed case studies for each project
- Live demo links and GitHub repository access
- Visual previews with optimized images

#### Development Resources

- Curated list of useful tools and resources
- Categorized sections for easy navigation
- Regular updates with new findings

#### Contact System

- Form validation using Zod
- EmailJS integration for serverless email handling
- Loading states and error handling
- Success notifications using Sonner

#### Technical Implementation

The website implements several modern web development practices:

- **App Router**: Utilizing Next.js 15's app router for improved routing and layouts
- **Dark Mode**: System-aware theme switching with next-themes
- **Type Safety**: Full TypeScript implementation for reliable code
- **Component Library**: Custom UI components built on shadcn/ui
- **Animation**: Subtle animations using Motion for enhanced UX
- **Performance**: Image optimization, code splitting, and lazy loading
- **Accessibility**: ARIA labels, keyboard navigation, and semantic HTML
